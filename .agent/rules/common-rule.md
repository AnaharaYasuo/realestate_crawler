# Common Rules

- `implementation_plan.md` は都度日本語で出力する。
- `walkthrough.md` の作成時には日本語で出力する。
- DBのテーブルやデータ削除は事前確認の実施を必須とする。
- 作業用のコードは Temp フォルダに出力する。
- ブラウザを開いて対象のサイトを読み込む作業をする場合、作業が終わったらそのタブを閉じること。
- 実装や設計の際は `docs/` 配下のドキュメント（仕様書、DBスキーマ、サイト構造など）を常に参照し、最新の仕様に準拠する。
- コードを修正したら必ずテストを実行し、全て成功するようにする
