# 収益不動産サイト 物件種別一覧

各収益不動産サイトで登録されている物件種別を調査した結果をまとめました。

> [!IMPORTANT]
> **本ドキュメントの位置づけ**
> 
> このドキュメントは、実装方針を決定するための事前調査結果です。各サイトで提供されている全物件種別を網羅的に記載していますが、実際の収集対象は以下の3種類に限定されます：
> 
> - **一棟マンション**（RC造・鉄骨造などの集合住宅一棟）
> - **一棟アパート**（木造・軽量鉄骨造などの集合住宅一棟）
> - **投資用戸建て**（賃貸収入目的の一戸建て）※サイトにより対応状況が異なる
> 
> 区分マンション、ビル、店舗/事務所、工場/倉庫、ホテル/旅館などは収集対象外です。

---

## 1. 三井のリハウス (Mitsui Rehouse)
**URL**: https://www.rehouse.co.jp/buy/tohshi/prefecture/13/city/13120/

### 物件種別

#### 投資・事業用不動産（主な建物種別）
- **一棟ビル**: ビル一棟全体
- **一棟マンション**: RC造などの大型集合住宅一棟全体
- **アパート（一棟アパート）**: 主に木造や軽量鉄骨造の集合住宅
- **店舗・事務所**: 主に事業用の店舗やオフィススペース
- **店舗・事務所（区分所有）**: ビルなどの一部フロアや区画を区分所有するもの
- **その他**: 工場、倉庫、駐車場、店舗付住宅など

#### その他の関連種別（投資・収益目的での利用を含む）
- **中古マンション（区分マンション）**: 区分所有の分譲マンション一戸単位（オーナーチェンジ物件など）
- **土地（売地）**: アパート用地やビル用地としての投資用土地
- **一戸建て（中古戸建）**: 一軒家を賃貸に回す投資用物件
- **新築マンション / 新築一戸建て**: 新築の収益物件

### 特徴
三井のリハウスでは、大きく「一棟もの（マンション・アパート・ビル）」、「区分所有（店舗・事務所・マンション）」、「土地・一戸建て」および「事業用（その他）」に分かれています。特に投資家向けには**一棟マンション、一棟アパート、一棟ビル**が独立したカテゴリーとして重視されています。

## 2. 住友不動産ステップ (Sumitomo Step)
**URL**: https://www.stepon.co.jp/search/list/?type=pro2&searchType=area&smk=111111111110&ca=0_001

### 物件種別（全12種類）

1. **マンション（一棟）**: マンション・アパート一棟全体
2. **ビル（一棟）**: オフィスビルや商業ビル一棟全体
3. **アパート（一棟）**: 木造・軽量鉄骨造の集合住宅一棟
4. **マンション（区分）**: 投資用の区分マンション（一室単位）
5. **店舗・事務所**: 店舗やオフィスなどの商業スペース
6. **投資用戸建**: 賃貸収入目的の一戸建て住宅
7. **工場・倉庫**: 工場や倉庫などの産業用不動産
8. **社宅・寮**: 企業の社宅や学生寮
9. **ホテル・旅館**: ホテルや旅館などの宿泊施設
10. **リゾート・別荘**: リゾート地の物件や別荘
11. **その他建物**: 上記に該当しないその他の建物
12. **土地**: 開発用地や投資用の土地

### 特徴
住友不動産ステップは最も細かく物件種別を分類しており、**12種類**の選択肢があります。ホテル・旅館、リゾート・別荘、社宅・寮など、他のサイトにはない特殊な物件種別も扱っています。

## 3. 野村不動産ソリューションズ (Nomu Pro)
**URL**: https://www.nomu.com/pro/all/area/11203/

### 物件種別（全11種類）

1. **一棟マンション**: マンション・アパート一棟全体（RC造・鉄骨造）
2. **一棟ビル**: オフィスビルや商業ビル一棟全体
3. **アパート / 一棟アパート**: 木造・軽量鉄骨造の集合住宅一棟
4. **区分マンション**: 区分所有の分譲マンション一室
5. **店舗・事務所**: 店舗やオフィスなどの商業用スペース
6. **投資用戸建て**: 投資目的の一戸建て住宅
7. **工場・倉庫**: 工場や倉庫などの産業施設
8. **ホテル・旅館**: ホテルや日本旅館
9. **寮・社宅**: 寮や企業の社宅
10. **その他建物**: その他の建物・構造物
11. **土地**: 開発用地や投資用の土地

### 特徴
野村不動産は住友不動産ステップとほぼ同様の分類を採用していますが、「リゾート・別荘」カテゴリがない点が異なります。全体的にバランスの取れた分類体系です。

## 4. 東急リバブル (Tokyu Livable)
**URL**: https://www.livable.co.jp/fudosan-toushi/tatemono-bukken-all/

### 物件種別（全9種類）

1. **マンション区分**: 区分所有の投資用マンション一室
2. **マンション一棟**: マンション一棟全体
3. **アパート**: アパート一棟全体（木造・軽量鉄骨造）
4. **ビル一棟**: オフィスビルや商業ビル一棟全体
5. **店舗・事務所**: 店舗やオフィスなどの商業用区分
6. **工場・倉庫**: 工場や倉庫などの産業施設
7. **社宅・寮**: 企業の社宅や寮
8. **その他建物 / その他物件**: その他の建物や特殊物件
9. **土地**: 投資用・開発用の土地

### 特徴
東急リバブルは**9種類**と比較的シンプルな分類です。ホテル・旅館やリゾート物件は独立したカテゴリとしては設けられておらず、「その他建物」に含まれると考えられます。

## 5. ミサワホーム不動産 (Misawa Real Estate)
**URL**: https://realestate.misawa.co.jp/search/sale/list/?bukken_type[]=9

### 物件種別

#### 主要な物件種別（検索フィルター上の分類）
- **土地**: 住宅用地や事業用地などの更地
- **一戸建て**: 一般的な中古の一戸建て住宅
- **ミサワホームの中古戸建**: 自社ブランド（ミサワホーム施工）の中古一戸建て
- **マンション**: 分譲マンションの区分所有物件
- **投資物件・その他**: 収益を目的とした不動産全般

#### 「投資物件・その他」カテゴリに含まれる具体的な種別
- **アパート**: 一棟アパート（木造・軽量鉄骨造などの集合住宅）
- **マンション**: 投資用の区分マンション、または一棟マンション
- **土地**: 収益化を前提とした事業用・投資用の土地
- **店舗付住宅**: 店舗と住居が一体となった物件

#### その他の特殊な投資形態
- **小口不動産投資**: 「ミサワスマートファンド」として展開されている不動産の特定共同事業（一棟不動産を小口化して投資する商品）

### 特徴
ミサワホーム不動産では、自社施工の特性を活かした「**ミサワホーム施工の中古戸建**」という独自のカテゴリがあります。また、不動産小口化商品（クラウドファンディング型）も扱っている点が特徴的です。物件種別の分類は他社と比べてやや大まかで、「投資物件・その他」という包括的なカテゴリにまとめられています。

### 主な傾向
1. **共通する基本カテゴリ**: すべてのサイトで「一棟マンション」「一棟アパート」「区分マンション」「土地」は共通して扱われています。
2. **最も詳細な分類**: **住友不動産ステップ**が12種類と最も細かく分類しています。
3. **特殊物件の扱い**: ホテル・旅館、リゾート物件、社宅・寮などの特殊物件は、住友不動産ステップと野村不動産が積極的に扱っています。
4. **独自カテゴリ**: ミサワホーム不動産の「ミサワホーム施工の中古戸建」「小口不動産投資」、三井のリハウスの「店舗・事務所（区分所有）」など、各社独自のカテゴリも存在します。
