# サイト構造: 住友不動産「ステップ」 (投資用)

## 概要
- **サイト名:** 住友不動産ステップ 投資用
- **ベースURL:** `https://www.stepon.co.jp/pro/`
- **タイプ:** 投資用不動産 (区分マンション, 一棟マンション, アパート, ビル, 土地 など)

## URLパターン

### 1. 検索結果 (一覧ページ)
- **検索ベースURL:** `https://www.stepon.co.jp/pro/search/list/`
- **エリア検索例:** `https://www.stepon.co.jp/pro/ca_0_001/?smk=[BITMASK]`
    - `ca_0_001`: エリアコード (0_001 = 全国)
    - `smk`: 物件種別ビットマスク (12桁)

#### 物件種別 (smk パラメータ)
| 種別名 | ビットマスク (`smk`) |
| :--- | :--- |
| **建物全て** | `111111111110` |
| **マンション (区分)** | `000100000000` |
| **マンション (一棟)** | `100000000000` |
| **アパート (一棟)** | `001000000000` |
| **ビル (一棟)** | `010000000000` |
| **土地** | `000000000001` |
| 店舗・事務所 | `000010000000` |
| 戸建 (投資用) | `000001000000` |

### 2. 詳細ページ
- **URL:** `https://www.stepon.co.jp/pro/detail_[PROPERTY_ID]/`

## セレクタ (詳細ページ)
`SumifuInvestment` モデルに対応する全フィールドの抽出定義。

### 共通ヘッダー・基本情報
| 項目名 (HTML/Concept) | モデルフィールド | セレクタ/取得元 |
| :--- | :--- | :--- |
| **物件名** | `propertyName` | `.property-info-name` or `h1` |
| **価格** | `price`, `priceStr` | `.property-info-price__num`, `.property-info-price__unit` |
| **URL** | `pageUrl` | Current URL |

### 投資情報ブロック (Investment Info)
ページ上部の「投資情報」セクション。

| 項目名 | モデルフィールド | 備考 |
| :--- | :--- | :--- |
| **想定利回り** | `grossYield` | `.investment-yield__num` (%) |
| **想定年間収入** | `annualRent` | 値をパース (万円/年) |
| **月額賃料** | `monthlyRent` | 計算または別途記載がある場合 |
| **現況** | `currentStatus` | 賃貸中, 満室, 空室など |

### 物件概要テーブル (`.table-detail`)
`table.table-detail` 内の `th` テキストに基づくマッピング。

#### 所在・交通
| 項目名 (th) | モデルフィールド | 備考 |
| :--- | :--- | :--- |
| **所在地** | `address` | |
| **交通** | `traffic` | 複数路線ある場合は連結 |
| **土地権利** | `landRights` | 所有権, 借地権など |

#### 建物・土地詳細
| 項目名 (th) | モデルフィールド | 備考 |
| :--- | :--- | :--- |
| **土地面積** | `landArea` | ㎡ |
| **建物面積** | `buildingArea` | ㎡ (区分マンションの場合は専有面積) |
| **専有面積** | `buildingArea` | 区分の場合の表記 |
| **築年月** | `yearBuilt` | |
| **階数 / 構造** | `structure` | RC, 鉄骨造など |
| **総戸数** | `totalUnits` | 一棟ものの場合 |
| **間取り** | `floorPlan` | 区分/戸建の場合 |
| **バルコニー面積** | `balconyArea` | 区分の場合 |
| **管理費等** | `managementFee` | 区分の場合 |
| **修繕積立金** | `repairReserve` | 区分の場合 |
